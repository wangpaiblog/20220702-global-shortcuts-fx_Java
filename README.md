分支 M # 版本 `M.5.0` 支持的功能：

1. 支持如下功能的全局快捷键：
   1. 程序可后台运行，无需依赖界面来维持
   2. 快捷键不会因为光标及界面切换而失效
   3. 支持重复快捷键。支持对某些不冲突的行为设置同一个快捷键
   4. 支持创建多实例。支持在代码创建本框架的实体类的多个实例，以独立运行多个任务
   
2. 本框架拥有可由快捷键控制的运行时生命周期，使用时只需要将快捷键、快捷键要触发的操作等，以 IoC 注入的方式注入到软件的不同生命周期后即可运行。

   故可以通过快捷键来控制本程序的启动、暂停、终止
